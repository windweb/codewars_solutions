## Convert a Number to a String!

**Description:**

We need a function that can transform a number (integer) into a string.

What ways of achieving this do you know?

**Examples (input --> output):**

```python
123  --> "123"
999  --> "999"
-100 --> "-100"
```
**Solution:**

```python
def number_to_string(num):
    return str(num)
```
